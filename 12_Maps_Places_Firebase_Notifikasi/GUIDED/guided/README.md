# Praktikum 12 - Maps and Places (GUIDED ONLY)

Ini project template **Guided** sesuai modul Praktikum 12:
- Google Maps (google_maps_flutter)
- Hybrid Composition (Android)
- myLocationEnabled (butuh permission & API key)

## Cara pakai cepat
1) Extract ZIP
2) Jalankan:
   flutter pub get
   flutter run
